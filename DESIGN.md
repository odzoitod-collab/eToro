# Дизайн биржи SellBit

Документ описывает визуальный стиль платформы SellBit и гайдлайны логотипа.

---

## 1. Общий дизайн интерфейса

### 1.1 Направление
- **Mobile-first**: интерфейс в первую очередь рассчитан на мобильные устройства и Telegram Mini App.
- **Тёмная тема**: глубокий чёрный фон для премиальности и комфорта при длительном использовании.
- **Акцент неонового зелёного**: используется для кнопок, иконок, состояний «успех» и ключевых действий.

### 1.2 Цветовая палитра UI

| Назначение        | Значение        | Использование |
|-------------------|-----------------|---------------|
| Фон основной     | `#050505`       | Экраны, подложка (почти чёрный, OLED-friendly) |
| Фон карточек     | `#0a0a0a`       | Блоки, инпуты, списки |
| Акцент (неон)    | Салатовый/неон  | Кнопки, иконки, границы при фокусе, свечение |
| Текст основной   | `white`         | Заголовки, важный текст |
| Текст вторичный  | `neutral-400`   | Подписи, второстепенный текст |
| Текст приглушённый | `neutral-500` / `neutral-600` | Подсказки, футер |
| Границы          | `neutral-800`   | Разделители, обводки карточек |
| Успех / рост     | `green-500`     | Положительные изменения цены, успешные действия |
| Ошибка / падение  | `red-500`       | Отрицательные изменения, ошибки |

Акцентный цвет в коде (Tailwind): `neon` — ярко-салатовый, в тени используется `rgba(163, 230, 53, …)` для эффекта свечения.

### 1.3 Типографика и отступы
- **Шрифты**: без засечек (гротеск), моноширинный (`font-mono`) для чисел, сумм и тикеров.
- **Скругления**: `rounded-xl` для карточек и кнопок, `rounded-2xl` для крупных блоков.
- **Отступы**: плотная сетка (например, `p-4`, `px-4`, `gap-2`/`gap-4`) для компактного, но читаемого интерфейса.

### 1.4 Ключевые паттерны
- Кнопки основного действия: фон `bg-neon`, текст чёрный, при наведении/нажатии — лёгкое свечение и `active:scale-[0.98]`.
- Карточки и инпуты: фон `#0a0a0a`, граница `border-neutral-800`, при фокусе/ховере — `border-neon/50`.
- Фоновые «блики»: крупные размытые круги с `bg-neon/5` и `blur-[120px]` для глубины без отвлечения.

## 1.5 Система интерфейса биржи (навигация, страницы, кнопки, тени, шрифты)

### Подход
- **Единый слой (Layout)**: все основные экраны рендерятся внутри `Layout`: фон, фоновые блики, сайдбар (десктоп), контентная зона, нижнее меню (мобильные). Навигация скрывается только на страницах KYC, CURRENCY, LANGUAGE и при открытой клавиатуре или модалках (например, вывод из стейкинга).
- **Контент**: `main` с `max-w-md lg:max-w-4xl xl:max-w-5xl 2xl:max-w-6xl mx-auto`, `overflow-y-auto`, нижний отступ `pb-32 lg:pb-8` при видимом меню (чтобы контент не уходил под навигацию).
- **Фон**: `bg-background` (Tailwind: `#050505`), поверх — два декоративных блика с `bg-neon/5`, `bg-neon/3`, `blur-[120px]` / `blur-[100px]` (z-0, pointer-events-none).

### Навигация

**Нижнее меню (мобильные/планшеты, `lg:hidden`)**  
- Стиль «полуостров»: прижато к низу экрана (`bottom-0`), закругление только сверху `rounded-t-2xl`.  
- Фон: `bg-[#0a0a0a]/98`, границы `border-t border-x border-white/10`, `backdrop-blur-xl`.  
- Тень: `shadow-[0_-4px_20px_rgba(0,0,0,0.3)]` — лёгкая тень вверх.  
- Высота зоны кнопок: `h-[50px]`, иконки 20px, подписи `text-[9px] font-medium tracking-wide`, активный пункт — `text-neon`, неактивный — `text-neutral-500` / `text-neutral-600`.  
- Отступ снизу: `paddingBottom: max(8px, env(safe-area-inset-bottom))`.  
- Пункты: **HOME**, **COINS**, **TRADING**, **EXCHANGE**, **DEALS** (иконки: Home, Coins, BarChart2, ArrowLeftRight, Briefcase).

**Сайдбар (десктоп, `hidden lg:flex`)**  
- Ширина `w-56 min-w-[14rem]`, фон `bg-[#050505]/95`, граница `border-r border-white/10`, `backdrop-blur-sm`.  
- Пункты те же пять; активный: `bg-neon/15 text-neon`, неактивный: `text-neutral-400 hover:text-white hover:bg-white/5`.  
- Кнопки: `px-4 py-3 rounded-xl`, иконка 22px, подпись `text-sm font-medium`.

### Доступность страниц
- **В нижнем меню/сайдбаре**: HOME, COINS, TRADING, EXCHANGE, DEALS.  
- **Без нижнего меню** (полноэкранные или отдельные флоу): KYC, CURRENCY, LANGUAGE; также меню скрывается при открытой клавиатуре и при открытой модалке вывода из стейкинга.  
- **Остальные страницы** открываются по переходам: DEPOSIT, WITHDRAW, QR_SCANNER, PROFILE — из главной (Quick Actions, хедер); KYC — из профиля; CURRENCY, LANGUAGE — из профиля/настроек. TRADING открывается с выбором актива с COINS или HOME.

### Кнопки
- **Основное действие (CTA)**: `bg-neon text-black font-bold`, скругление `rounded-xl`, высота обычно `py-3` / `py-3.5`, `active:scale-[0.98]`, при необходимости `shadow-[0_0_20px_rgba(163,230,53,0.2)]`. Отключённое состояние: `disabled:opacity-50 disabled:cursor-not-allowed`.  
- **Вторичная / обводка**: `border border-white/20` или `border-neon/50`, фон `bg-white/5` или прозрачный, `hover:bg-neon/10`, `hover:border-neon/30`, `text-neutral-300` / `text-neon`.  
- **Переключатели (табы, направление обмена)**: активный — `bg-neon/20 text-neon border border-neon/40`, неактивный — `text-neutral-500 hover:text-neutral-300`.  
- **Быстрые действия на главной**: круглые кнопки `h-14 w-14 lg:h-16 lg:w-16 rounded-full`; главная (Пополнить) — `bg-neon text-black shadow-[0_0_15px_rgba(163,230,53,0.3)]`; остальные — `bg-neutral-900 border-white/5`.  
- **Хаптик**: на клики по навигации и кнопкам вызывается `Haptic.tap()` (или `success`/`error` по результату).

### Карточки и блоки
- Фон: `bg-[#0a0a0a]`, граница: `border border-white/10` или `border-neutral-800`, скругление `rounded-xl` (реже `rounded-2xl` для крупных блоков).  
- При наведении/фокусе: `hover:border-neon/50`, иногда `focus:border-neon/50 focus:ring-1 focus:ring-neon/50`.  
- Липкие шапки (например, поиск на Рынке): `sticky top-0 z-50 bg-[#050505]` и при необходимости `shadow-[0_10px_30px_rgba(0,0,0,0.8)]`.

### Поля ввода
- Контейнер: `bg-[#0a0a0a]` или `bg-[#050505]`, `border border-neutral-800`, `rounded-xl`, `focus-within:border-neon/50`.  
- Текст: `text-white font-mono`, размер `text-sm` / `text-base` / `text-lg` в зависимости от экрана, placeholder `placeholder-neutral-600` / `placeholder-neutral-700`.

### Тени и подсветки
- **Неон-подсветка кнопок**: `shadow-[0_0_10px_rgba(163,230,53,0.3)]`, `shadow-[0_0_20px_rgba(163,230,53,0.2)]`, для подчёркивания таба — `shadow-[0_0_8px_#a3e635]`.  
- **Фоновые блики**: `bg-neon/5`, `bg-neon/3`, `blur-[120px]`, `blur-[100px]` (в Layout).  
- **Нижнее меню**: тень вверх `shadow-[0_-4px_20px_rgba(0,0,0,0.3)]`.  
- **Хедер главной**: `shadow-md shadow-black/50`.  
- **Чарт/виджет**: тонкая обводка и внутренняя линия `shadow-[0_0_0_1px_rgba(163,230,53,0.06),inset_0_1px_0_rgba(255,255,255,0.03)]`.

### Размеры шрифтов
- **Заголовки страниц**: `text-xl font-bold text-white`.  
- **Крупные суммы/тикеры**: `text-lg font-mono font-bold text-neon` или `text-white`.  
- **Подзаголовки / лейблы**: `text-xs` / `text-[11px]` / `text-[10px]`, часто `uppercase tracking-wide text-neutral-500`.  
- **Основной текст**: `text-sm`, для чисел — `font-mono`.  
- **Мелкий текст / подписи**: `text-[10px]` / `text-[9px]`, `text-neutral-500` / `text-neutral-400`.  
- **Навигация**: подписи `text-[9px]` (низ), `text-sm` (сайдбар).

### Анимации
- Появление страниц: `animate-fade-in` (opacity + translateY 8px, 0.2s ease-out).  
- Модалки и оверлеи: те же классы при необходимости, z-index 60–200 в зависимости от слоя (меню 50, модалки 100, пин/онбординг 200).  
- Кнопки: `active:scale-95` / `active:scale-[0.98]`, `transition-transform` / `transition-all duration-200`.
- **Переходы модалок и вкладок**: `cubic-bezier(0.4, 0, 0.2, 1)` (класс `transition-modal`, переменная `--ease-modal` в `index.html`).
- **Order Book Flash**: при изменении цены — мягкая подсветка зелёным (рост) или красным (падение) с затуханием 300ms (классы `animate-flash-up`, `animate-flash-down`).
- **Hover**: строки таблиц — `hover-row` (лёгкий фон), кнопки CTA — `hover-glow` (тонкое свечение границы).

### Сетка отступов и CSS-переменные (торговый экран)
- **Базовая сетка**: 4px / 8px (в Tailwind: `p-1` = 4px, `p-2` = 8px, `p-4` = 16px; в `index.html` добавлены токены `panel`, `block`, `table`).
- **Визуальная иерархия**: внешние отступы между блоками (стакан, график, форма) — 16–24px (`gap-4` / `gap-6`, `mt-6`); внутренние отступы в таблицах стакана — минимальные (`py-[2px]`, `px-2`) для плотности данных.
- **Заголовки панелей**: больше «воздуха» — `py-3` в шапке стакана, отступ между секциями формы `gap-4`.
- **Переменные в `:root`** (в `<style>` в `index.html`): `--ease-modal`, `--duration-flash`, `--duration-modal`, `--color-flash-up`, `--color-flash-down`, `--border-subtle`, `--border-beam`. Учитывается `prefers-reduced-motion: reduce`.
- **Пример на чистом CSS**: ключи `flash-up` / `flash-down` и классы `.animate-flash-up` / `.animate-flash-down`. Для Framer Motion: модалка — `transition={{ duration: 0.28, ease: [0.4, 0, 0.2, 1] }}`, flash — `animate={{ backgroundColor: [color, 'transparent'] }}` с duration 0.3.

### Тема Tailwind (index.html)
- **Цвета**: `background: '#050505'`, `surface: '#0a0a0a'`, `neon: '#a3e635'`, `subtle: '#1f1f1f'`.  
- **Шрифты**: `sans: ['Inter']`, `mono: ['Roboto Mono']`.

## 2. Логотип SellBit

### 2.1 Цветовая палитра логотипа

- **Основной цвет**: ярко-салатовый (неоновый зелёный).  
  Ассоциации: рост, энергия, обновление, цифровая среда. Хороший контраст с фоном и эффект «свечения».

- **Фон**: глубокий чёрный (`#050505` или темнее).  
  Премиальность, строгость, отличная читаемость на экранах (в т.ч. OLED).

### 2.2 Графический знак (эмблема)

- **Расположение**: над текстовой частью («SellBit»).

- **Идея**: абстрактная стилизация первой буквы названия — **«S»**.

- **Геометрия**:
  - Буква «S» состоит из двух ломаных сегментов.
  - **Верхняя часть**: напоминает молнию или быстрый росчерк.
  - **Нижняя часть**: острый угол, направленный вниз — алмаз (бриллиант) или наконечник стрелы.

- **Динамика**: слева у эмблемы — острый «хвост», выходящий за основные границы. Создаёт ощущение скорости и мгновенных транзакций.

- **Объём**: за счёт разрывов между линиями и разной толщины штрихов — иллюзия многослойности при плоском (flat) исполнении.

### 2.3 Типографика (слово «SellBit»)

- **Расположение**: под эмблемой.

- **Начертание**: современный гротеск (без засечек) со скруглёнными углами. Смягчает острый характер эмблемы и делает бренд более дружелюбным.

- **Акценты**:
  - **«S»** — заглавная, высокая.
  - **«B»** — тоже заглавная (CamelCase: Sell + Bit), что помогает сразу прочитать два слова: «Sell» (продавать) и «Bit» (бит/биткоин).
  - **«t»** — верхняя часть срезана под углом, перекликается с острыми углами графического знака.

- **Межбуквенное расстояние**: умеренное, обеспечивает читаемость при сильном уменьшении (в т.ч. иконка приложения).

### 2.4 Композиция и символика

- **Симметрия**: центральная осевая композиция — устойчивость и доверие (важно для финансовой/криптобиржи).

- **Скрытые смыслы**:
  - Форма «S» в сочетании с ромбовидным основанием намекает на **безопасность** (щит) и **ценность** (кристалл/актив).

### 2.5 Использование логотипа

- На тёмном фоне: эмблема и текст — неоновый зелёный по чёрному.
- Минимум использования: только эмблема для фавикона и иконок приложения.
- Запрещено: искажение пропорций, замена цвета на произвольный без утверждения, наложение на пёстрый фон без контрастной подложки.

## 3. Связь логотипа и UI

- Цвет эмблемы и акцентный цвет интерфейса совпадают (неоновый зелёный).
- Острые углы эмблемы поддерживаются острыми акцентами в UI (например, срез у «t», углы кнопок при необходимости).
- Общий стиль: строгий, цифровой, быстрый — в духе биржи с мгновенными сделками и современным интерфейсом.

## 4. Промт: анимированный SVG-баннер «Нет активных сделок»

Используй этот промт для генерации или отрисовки **анимированного SVG** в стиле биржи SellBit — для пустого состояния вкладки «Активные» на странице портфеля. Анимация должна быть в духе гифа: плавная, зацикленная, без тяжёлых эффектов.

### 4.1 Промт для дизайнера / ИИ

```
Создай анимированный SVG-баннер для экрана пустого портфеля криптобиржи.

Тема: «У вас нет активных сделок» — спокойное, дружелюбное приглашение открыть первую сделку, без ощущения ошибки.

Стиль:
- Тёмный фон: глубокий чёрный (#050505 или прозрачный для встройки в #0a0a0a).
- Единственный яркий акцент: неоновый салатовый/лайм (#A3E635 или rgb(163, 230, 53)), возможны полупрозрачные варианты для свечения (rgba(163, 230, 53, 0.3–0.6)).
- Вторичные элементы: нейтральные серые (neutral-500/600), тонкие линии, без перегруза.
- Flat-дизайн, минимализм, острые углы и чёткие формы в духе логотипа (молния, стрела, кристалл).

Сюжет и элементы (на выбор или комбинация):
- Абстрактная «S» или молния в стиле логотипа SellBit — мягкое пульсирующее свечение или лёгкое движение по траектории (как будто ток/сигнал).
- Пустой портфель/кошелёк: силуэт сумки или портфеля с тонкой неоновой обводкой, внутри — тихий градиент или частицы, намекающие на «ожидание сделок».
- Кристалл/алмаз (символ ценности): контур с неоновой подсветкой, лёгкое мерцание граней или пульс по контуру.
- График/свеча: одна-две минималистичные свечи без агрессии — статичные или с очень мягким появлением/исчезновением.
- Точки или пунктирная линия: «ожидание ввода», как скелетон загрузки — плавное появление/исчезновение по очереди (stagger).

Анимация (в духе гифа, зацикленная):
- Длительность цикла: 2–4 секунды.
- Эффекты: opacity, stroke-dashoffset, transform (scale, translate), filter: drop-shadow/glow для неона.
- Без резких скачков: ease-in-out, плавные ключевые кадры.
- Можно: лёгкое дыхание (pulse) неонового контура, бегущая точка по пути, мерцание одной грани кристалла, появление пунктира «слева направо».
- Не перегружать: 1–2 главных движения, остальное статика или очень тонкое движение.

Технические требования:
- SVG viewBox удобный для баннера (например 320x160 или 400x200), соотношение сторон под карточку в мобильном UI.
- Все анимации — через CSS animation или SMIL <animate>/<animateTransform>, чтобы можно было встроить в React/HTML и при необходимости отключить (prefers-reduced-motion).
- Цвета только из палитры: #050505, #0a0a0a, #A3E635, нейтральные серые (#737373, #525252). Никаких других ярких цветов.
- Текст «У вас нет активных сделок» в баннер можно не включать — он будет отдельно в интерфейсе под/над SVG. Если включаешь: шрифт без засечек, цвет neutral-400 или белый, мелкий кегль.
```

### 4.2 Краткий промт (одним абзацем)

**Копия для быстрой вставки:**

Анимированный SVG для баннера «Нет активных сделок» в портфеле криптобиржи: тёмный фон #050505, акцент неоновый зелёный #A3E635, flat minimal style. Сюжет — стилизованная молния/«S» в духе логотипа или пустой портфель/кристалл с неоновой обводкой. Плавная зацикленная анимация 2–4 сек: пульс свечения, лёгкое движение по траектории или мерцание граней; только CSS/SMIL, без лишних цветов. Соотношение под карточку ~320x160, минимализм, как гиф.

### 4.3 Где использовать

- **Страница**: `DealsPage` (Портфель сделок), вкладка «Активные».
- **Условие**: `activeTab === 'ACTIVE' && activeDeals.length === 0`.
- **Вместо**: текущего блока с иконкой `Timer` и текстом `no_active`.
- **Размер**: под высоту баннера ~120–200px, ширина на всю карточку (минус отступы). SVG масштабировать через `width="100%"` и `height="auto"` или фиксированную высоту с `preserveAspectRatio="xMidYMid meet"`.
